# Bimbel Prestasi 🎓

Landing page untuk lembaga bimbingan belajar **Bimbel Prestasi**. Website ini dirancang untuk memberikan informasi mengenai keunggulan, program belajar, testimoni siswa, dan formulir pendaftaran.

## 🚀 Fitur
- **Beranda**: Halaman utama dengan hero section yang menarik.
- **Keunggulan**: Informasi mengapa memilih Bimbel Prestasi.
- **Program**: Detail program belajar yang ditawarkan.
- **Testimoni**: Pengalaman dari siswa-siswa sebelumnya.
- **Kontak & Daftar**: Halaman untuk menghubungi dan mendaftar.

## 🛠️ Teknologi
- HTML5
- CSS3
- Bootstrap 5
- Popper.js

## 📂 Struktur Proyek
```text
bimbel/
├── index.html              # Halaman Utama (Entry Point)
└── src/
    ├── asset/              # Gambar dan logo
    ├── css/                # File stylesheet (CSS)
    └── uts/                # Halaman pendukung (Keunggulan, Program, dll)
```

## ⚙️ Setup & Cara Menjalankan

Project ini menggunakan Bootstrap sebagai framework CSS yang dikelola melalui npm.

### 1. Clone Repository
```bash
git clone https://github.com/AhmadYusronF/bimbel-desweb.git
cd bimbel
```

### 2. Install Dependencies
Instal paket Bootstrap dan dependensinya menggunakan npm:
```bash
npm install
```

### 3. Jalankan di Browser
Ada dua cara untuk membuka project ini:

**Cara A: Buka File Langsung**
- Buka folder project.
- Klik kanan pada file `index.html` dan pilih **Open with** -> **Browser (Chrome/Edge/Firefox)**.

**Cara B: Menggunakan Live Server (Rekomendasi)**
Jika Anda menggunakan VS Code:
1. Install ekstensi **Live Server**.
2. Klik kanan pada `index.html`.
3. Pilih **Open with Live Server**.
4. Website akan otomatis terbuka di `http://127.0.0.1:5500`.
